# dotfiles

## environment

These dotfiles are for debian based systems.
Mostly used on a Debian/WSL2 Installation.

## covered applications

    1. NeoVim
    2. LazyGit
    3. tmux
    4. spotify_player

## required applications

    1. jq for parsing JSON provided
    2. timeout to keep requests from killing tmux
    3. (optional) GNU Stow to copy/symlink all this to the required folders
